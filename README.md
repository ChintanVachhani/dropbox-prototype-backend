# dropbox-prototype-backend
A simple Dropbox application replica. Back-end using Spring Boot.

## Git Repository

```
  git clone https://github.com/ChintanVachhani/dropbox-prototype-backend.git
```

**Note : Please make sure your MongoDB. Also `npm3` is required to install dependencies properly.

## Quickstart

1. Run backend
   ```
    run java class - 'com.dropboxPrototypeBackend.DropboxPrototypeBackEndApplication'
   ```
2. Run frontend
   ```
   cd dropbox-prototype-frontend
   npm install
   npm start
